# OpenCraft Python Edition
## Controls
W,S,A,D - move around <br/>
Numbers - choose a block <br/>
RMB - place a block <br/>
LMB - destroy a block <br/>
F1 - show/hide the fps counter <br/>
Escape - quit <br/>
/ - open the command line
